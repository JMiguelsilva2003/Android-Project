# Android Project 💻

**Curiosidades de Tecnologia**: Um site que reúne tudo o que você sempre quis saber sobre o mundo tech em um único lugar. Descubra a história fascinante do mascote do Android e muito mais!

📱 **Visite o Projeto**: [Android Project](https://jmiguelsilva2003.github.io/Android-Project/index.html.html)

## 🎨 Layout
O site apresenta um design intuitivo com uma navegação fácil pelo menu, incluindo seções como Home, Notícias e Fale Conosco.

## 💻 Tecnologias
Este projeto foi desenvolvido utilizando:
- HTML
- CSS

## 🚀 Começando
Para executar o projeto localmente, siga estas etapas:

### Pré-requisitos
Certifique-se de ter instalado:
- Um navegador web atualizado (o site funciona melhor no Google Chrome)

### Clonando
Para clonar o projeto, execute o seguinte comando:
```
git clone https://github.com/jmiguelsilva2003/Android-Project.git
```

### Iniciando
Abra o arquivo `index.html.html` no seu navegador para visualizar o projeto.
